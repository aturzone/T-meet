- Phase 08 â€” E2E Chat: per-session X25519 ephemeral keypair via `libsodium-wrappers-sumo` (`crypto_box_keypair`), generated lazily on first call to `initSessionKeys()` and cached in module scope. `crypto_box_seal` + `crypto_box_seal_open` for per-recipient encryption â€” server only sees ciphertext + recipient hints. Backend protocol additions: `ClientMsg::Announce { pubkey }` + `ServerMsg::PeerUpdated { peer }`; the room hub gains `set_pubkey` to update a participant's descriptor and broadcast the change. Private key is `sodium.memzero`'d on leave. 8 new Vitest tests cover keys, seal/open round-trip, tamper detection, wrong-key, oversize plaintext, and the chat store. New `frontend/scripts/link-libsodium-sumo.mjs` postinstall script materializes the symlink `libsodium-wrappers-sumo` needs to resolve its sibling `libsodium-sumo.mjs` under pnpm.
