## Best-practice choices baked in

- **AGPL-3.0** matches the privacy-first, self-host ethos of the project.
- **MSRV pin** via `clippy.toml` + `rust-toolchain.toml` (stable, not nightly).
- **`cargo-deny`** in CI rejects copyleft surprises and bans `openssl-sys` (rustls-only invariant from prompt §3).
- **PASETO v4 local** over JWT (prompt §3) — simpler, fewer footguns, no `alg:none`.
- **`#![forbid(unsafe_code)]`** at every crate root — codified in Phase 00.
- **`rust-embed`** with `compress = true` for the frontend bundle.
- **`axum-server` + `rustls`** (not hyper-rustls directly) — clean musl link, no OpenSSL transitively.
- **`sqlx` with `runtime-tokio-rustls`** + bundled SQLite + offline mode (`sqlx prepare`) so CI builds don't need a DB.
- **`tracing`** with `EnvFilter`, no `println!`. JSON formatter in release.
- **Playwright via Brave** — matches sibling T-* projects, exercises real DTLS-SRTP + getUserMedia (Brave supports `--use-fake-ui-for-media-stream` + `--use-fake-device-for-media-stream` for headless WebRTC tests).
- **`pnpm`** with `packageManager` pin in `package.json` + Corepack for determinism.
- **Vitest** for unit, **Playwright** for E2E driving the actual server binary in CI.
- **Conventional commits scoped per phase**: `phase-NN: <area>: <verb> <thing>`.
- **`SECURITY.md` with private channel** — required for OSS security hygiene.
- **Reproducible-ish releases** via `SOURCE_DATE_EPOCH` + locked `Cargo.lock` and `pnpm-lock.yaml`.
